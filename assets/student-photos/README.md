# Thu Vien Anh Hoc Vien

Thu vien nay luu anh hoc vien that de chon them vao bai dang Facebook.

Quy uoc:

- Moi chu de la mot thu muc album, vi du `tin-hoc/`, `ky-thuat/`, `ke-toan/`, `tre-em/`.
- Moi album co file `photos.json` de khai bao metadata cho tung anh.
- Tat ca anh trong album deu co the duoc them vao bo anh dang bai.
- Nen dat ten file khong dau, khong khoang trang, vi du `lop-tin-hoc-001.jpg`.

Vi du metadata trong `photos.json`:

```json
{
  "photos": [
    {
      "id": "tin-hoc-001",
      "file": "lop-tin-hoc-001.jpg",
      "title": "Lop tin hoc van phong",
      "approved_for_use": true,
      "taken_at": "2026-06-01",
      "order": 1
    }
  ]
}
```
