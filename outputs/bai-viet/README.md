# Quy Ước Lưu Bài Viết

Mỗi bài viết được lưu trong một thư mục riêng theo tên bài viết dạng không dấu.

Ví dụ:

```text
outputs/bai-viet/
  tin-hoc-tre-em/
    bai-viet.md
    hinh-anh.png
```

Quy ước file:

- `bai-viet.md`: nội dung bài viết, caption, hashtag, prompt thiết kế nếu có.
- `hinh-anh.png`: ảnh chính của bài viết.
- `hinh-anh-2.png`, `hinh-anh-3.png`: các phiên bản ảnh khác nếu có.
